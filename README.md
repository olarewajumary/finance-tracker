# Finance Tracker

A Python app that reads your bank statement CSV and automatically 
visualizes your spending patterns with charts and summaries.

---

## Features

- Loads and cleans bank statement CSV files
- Generates a full spending summary report
- Shows total income, total spent and net balance
- Breaks down spending by category
- Visualizes daily spending over time
- Saves charts automatically to output folder

---

## Tech Stack

- Python 3.x
- Pandas
- Matplotlib
- Seaborn
